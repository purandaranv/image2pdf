# Image to PDF Converter

A simple, client-side web app that lets you upload an image and convert it into a downloadable PDF file.

## Features

- Upload an image by clicking or dragging and dropping it into the app
- Preview the selected image before converting
- Convert the image to an A4-sized PDF using jsPDF
- Download the generated PDF directly from the browser

## How to use

### Locally

1. Open [index.html](index.html) in your browser.
2. Click the upload area or drag an image onto it.
3. Preview the image and click "Convert to PDF".
4. The browser will download the generated PDF file.

### On GitHub Pages

1. Push the project to GitHub.
2. Open the repository Settings and enable GitHub Pages.
3. Set the source to GitHub Actions.
4. Wait for the deployment workflow to finish, then open your GitHub Pages URL.

## Requirements

- A modern web browser such as Chrome, Edge, Firefox, or Safari
- No installation or build step is required

## Notes

This app runs entirely in the browser. Your images are not uploaded to any server, and the site is designed to work as a static GitHub Pages project.
